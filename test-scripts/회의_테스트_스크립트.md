## 🔍 디버깅 체크리스트

브라우저 콘솔(F12)에서 다음 로그가 표시되는지 확인하세요:

1. `📝 분석할 transcript: [김프로] 안녕하세요...`
2. `✅ API 응답: {status: "success", data: {...}}`
3. `📊 최종 mergedData: {title: "AI 분석 완료된 회의", ...}`

**로그가 없다면?**
- `⚠️ transcript가 없습니다.` → localStorage에 저장이 안 됨
- 네트워크 탭에서 `/api/analyze-meeting` 요청 상태 확인
